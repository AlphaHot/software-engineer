# Bit Operations
Name | Operation
--- | ---
set union | A \| B
set intersection | A & B
set subtraction  | A & ~B
set negation | ALL_BITS ^ A or ~A
set bit | A \|= 1 << bit
clear bit | A &= ~(1 << bit)
test bit | (A & 1 << bit) != 0
extract last bit | A&-A or A&~(A-1) or x^(x&(x-1))
remove last bit | A & (A-1)
get all | 1-bits ~0
-A | ~A + 1
